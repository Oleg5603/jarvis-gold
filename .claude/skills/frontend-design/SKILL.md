<h3 node="[object Object]">Design Thinking</h3>
<p>Before coding, understand the context and commit to a BOLD aesthetic direction:</p>
<ul>
<li><strong>Purpose</strong>: What problem does this interface solve? Who uses it?</li>
<li><strong>Tone</strong>: Pick an extreme: brutally minimal, maximalist chaos, retro-futuristic, organic/natural, luxury/refined, playful/toy-like, editorial/magazine, brutalist/raw, art deco/geometric, soft/pastel, industrial/utilitarian, etc. There are so many flavors to choose from. Use these for inspiration but design one that is true to the aesthetic direction.</li>
<li><strong>Constraints</strong>: Technical requirements (framework, performance, accessibility).</li>
<li><strong>Differentiation</strong>: What makes this UNFORGETTABLE? What&#x27;s the one thing someone will remember?</li>
</ul>
<p><strong>CRITICAL</strong>: Choose a clear conceptual direction and execute it with precision. Bold maximalism and refined minimalism both work - the key is intentionality, not intensity.</p>
<p>Then implement working code (HTML/CSS/JS, React, Vue, etc.) that is:</p>
<ul>
<li>Production-grade and functional</li>
<li>Visually striking and memorable</li>
<li>Cohesive with a clear aesthetic point-of-view</li>
<li>Meticulously refined in every detail</li>
</ul>
<h3 node="[object Object]">Frontend Aesthetics Guidelines</h3>
<p>Focus on:</p>
<ul>
<li><strong>Typography</strong>: Choose fonts that are beautiful, unique, and interesting. Avoid generic fonts like Arial and Inter; opt instead for distinctive choices that elevate the frontend&#x27;s aesthetics; unexpected, characterful font choices. Pair a distinctive display font with a refined body font.</li>
<li><strong>Color &amp; Theme</strong>: Commit to a cohesive aesthetic. Use CSS variables for consistency. Dominant colors with sharp accents outperform timid, evenly-distributed palettes.</li>
<li><strong>Motion</strong>: Use animations for effects and micro-interactions. Prioritize CSS-only solutions for HTML. Use Motion library for React when available. Focus on high-impact moments: one well-orchestrated page load with staggered reveals (animation-delay) creates more delight than scattered micro-interactions. Use scroll-triggering and hover states that surprise.</li>
<li><strong>Spatial Composition</strong>: Unexpected layouts. Asymmetry. Overlap. Diagonal flow. Grid-breaking elements. Generous negative space OR controlled density.</li>
<li><strong>Backgrounds &amp; Visual Details</strong>: Create atmosphere and depth rather than defaulting to solid colors. Add contextual effects and textures that match the overall aesthetic. Apply creative forms like gradient meshes, noise textures, geometric patterns, layered transparencies, dramatic shadows, decorative borders, custom cursors, and grain overlays.</li>
</ul>
<p>NEVER use generic AI-generated aesthetics like overused font families (Inter, Roboto, Arial, system fonts), cliched color schemes (particularly purple gradients on white backgrounds), predictable layouts and component patterns, and cookie-cutter design that lacks context-specific character.</p>
<p>Interpret creatively and make unexpected choices that feel genuinely designed for the context. No design should be the same. Vary between light and dark themes, different fonts, different aesthetics. NEVER converge on common choices (Space Grotesk, for example) across generations.</p>
<p><strong>IMPORTANT</strong>: Match implementation complexity to the aesthetic vision. Maximalist designs need elaborate code with extensive animations and effects. Minimalist or refined designs need restraint, precision, and careful attention to spacing, typography, and subtle details. Elegance comes from executing the vision well.</p>
