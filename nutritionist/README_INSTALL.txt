========================================
  СУФЛЁР НУТРИЦИОЛОГА — Инструкция
========================================

КАК УСТАНОВИТЬ (первый раз):
-------------------------------
1. Скачай Python 3.11 с сайта python.org
   При установке поставь галочку "Add Python to PATH"

2. Положи все файлы из этой папки в одну директорию

3. Запусти SETUP_AND_BUILD.bat
   (правая кнопка → "Запустить от имени администратора")

4. Подожди 5-10 минут — установятся библиотеки и соберётся приложение

5. Готово! Приложение в папке:  dist\Sufler\Sufler.exe

========================================
КАК ПОЛЬЗОВАТЬСЯ:
========================================

1. Запусти Sufler.exe

2. Откроются два окна:
   - ЛЕВОЕ: "Что спросить" — вопросы по теме + таймер
   - ПРАВОЕ: "Рекомендации" — советы по Малаховой

3. Поставь длительность консультации (мин), нажми ▶ Старт

4. Говори с клиентом — суфлёр слушает через микрофон
   Услышал ключевое слово → оба окна обновятся автоматически

5. Можно перетаскивать окна, менять шрифт и размер текста

6. Кнопка "+10 мин" — добавить время к таймеру

========================================
ТРЕБОВАНИЯ:
========================================
- Windows 10 / 11
- Python 3.11+ (нужен только для первой сборки)
- Интернет (для распознавания речи через Google)
- Микрофон
