# Misemia — Бот психотерапевта Светланы

## О проекте

Telegram-бот для управления ВКонтакте страницей психотерапевта Светланы.  
ВК группа: [vk.ru/misemia](https://vk.ru/misemia) (ID: 227082800)

## Что умеет бот

- Генерирует посты по теме (через Claude AI) в стиле misemia
- Автоматически добавляет смайлики и уникальное ИИ-фото по теме (Pollinations.ai)
- В конце каждого поста контакт: `+79028355176 (мессенджер МАКС)`
- Публикует пост в ВК после одобрения клиентом кнопкой
- Проверяет и отвечает на комментарии в ВК (каждый час автоматически)

## Основные файлы

- `misemia_bot.py` — код бота
- `misemia_draft.json` — черновики постов
- `misemia_owner.txt` — chat_id владельца
- `misemia_landing/index.html` — лендинг психотерапевта

## Лендинг

- Лежит на сервере: `/root/telegram-bot/misemia_landing/index.html`
- Открывается по: `http://159.194.200.172:8081`
- Кнопки ведут на ВКонтакте и к Максу в Telegram

## Команды бота

| Команда | Действие |
|---------|----------|
| `/start` | Начало работы |
| `/post [тема]` | Сгенерировать пост |
| `/comments` | Проверить комментарии в ВК |

## Сервис на сервере

```bash
systemctl status misemia-bot
systemctl restart misemia-bot
```
