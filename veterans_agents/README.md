# Система агентов ветеранской организации

Multi-agent система на базе Claude AI для автоматизации работы ветеранской организации.

## Агенты

| Агент | Зона ответственности |
|-------|---------------------|
| **admin** | Делопроизводство, протоколы заседаний, отчётность, планирование |
| **social** | Соцпомощь, льготы, субсидии, волонтёры, соцзащита |
| **membership** | Приём членов, база данных, учёт первичных организаций |
| **events** | Мероприятия, патриотическая работа, памятные даты, захоронения |
| **comms** | СМИ, соцсети (ВК, ОК, Telegram), рассылки, пресс-релизы |

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
export ANTHROPIC_API_KEY=sk-ant-...
python main.py
```

## Использование

```
>> Составь протокол заседания президиума от 1 июля 2026 г.
# → автоматически направит к агенту admin

>> Нужно организовать раздачу продуктов 15 ветеранам перед 9 мая
# → агент social

>> Добавить нового члена: Иванов Пётр Семёнович, 1945 г.р., ул. Ленина 5 кв.12
# → агент membership

>> Подготовить план Урока мужества для 9-го класса
# → агент events

>> Написать пост ВКонтакте о предстоящем субботнике у мемориала
# → агент comms
```

### Прямой вызов агента

```
>> /admin Составь годовой план работы организации на 2026 год
>> /social Список документов для оформления субсидии на ЖКХ
```
