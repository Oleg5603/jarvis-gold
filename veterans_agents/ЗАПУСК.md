# Бот ветеранской организации — инструкция запуска

## Что нужно

- Python 3.10+
- Токен Telegram-бота (получить у @BotFather)
- Один ключ OpenRouter — даёт доступ ко всем моделям сразу (Llama, Claude, Gemini)
  - openrouter.ai/keys → Create Key

## Установка

```bash
pip install -r requirements_bot.txt
```

## Настройка

Задать переменные окружения:

```bash
# Windows (cmd)
set TELEGRAM_BOT_TOKEN=7xxxxxxxxxx:AAxxxxxxxxxxxxxxxx
set OPENROUTER_API_KEY=sk-or-v1-xxxxxxxxxxxxxxxx

# Windows (PowerShell)
$env:TELEGRAM_BOT_TOKEN="7..."
$env:OPENROUTER_API_KEY="sk-or-v1-..."

# Linux / Mac
export TELEGRAM_BOT_TOKEN=7...
export OPENROUTER_API_KEY=sk-or-v1-...
```

## Запуск

```bash
python bot_client.py
```

## Что умеет бот

Пользователь выбирает раздел кнопкой или пишет вопрос свободным текстом:

| Кнопка | Что делает |
|--------|-----------|
| 📋 Документы | Протоколы, отчёты, планы, приказы |
| 🤝 Соцпомощь | Льготы, субсидии, волонтёры |
| 👥 Члены | Приём, база данных, первички |
| 🎖️ Мероприятия | 9 мая, уроки мужества, захоронения |
| 📢 Инфо и СМИ | Посты ВКонтакте, пресс-релизы, рассылки |
| 🔀 Авто | Бот сам определяет нужного агента |
| ⚙️ Модель ИИ | Переключение между Llama / Claude / Gemini |

По умолчанию используется **Llama 3.3 (бесплатно)**. Переключить модель можно в любой момент кнопкой «⚙️ Модель ИИ» — выбор сохраняется для текущего диалога.

## Деньги

- **Llama 3.3 (`:free`)** — полностью бесплатно, есть лимит запросов в минуту/день на бесплатные модели OpenRouter
- **Claude Haiku / Gemini Flash** — платные, баланс пополняется в личном кабинете openrouter.ai (Credits), минимум обычно $5
- Один баланс на все модели — не нужно заводить отдельные кабинеты в Anthropic/Google
