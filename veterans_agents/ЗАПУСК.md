# Бот ветеранской организации — инструкция запуска

## Что нужно

- Python 3.10+
- Токен Telegram-бота (получить у @BotFather)
- Хотя бы один ключ ИИ-провайдера (можно несколько — переключаются прямо в боте):
  - **Claude** (платный) — console.anthropic.com
  - **Gemini** (есть бесплатный тариф) — aistudio.google.com/apikey
  - **Groq** (есть бесплатный тариф) — console.groq.com/keys

## Установка

```bash
pip install -r requirements_bot.txt
```

Если нужен только один провайдер — можно поставить не все библиотеки, а только нужную:

```bash
pip install python-telegram-bot anthropic          # только Claude
pip install python-telegram-bot google-generativeai  # только Gemini
pip install python-telegram-bot groq                # только Groq
```

## Настройка

Задать переменные окружения (нужен хотя бы один ключ ИИ-провайдера + токен Telegram):

```bash
# Windows (cmd)
set TELEGRAM_BOT_TOKEN=7xxxxxxxxxx:AAxxxxxxxxxxxxxxxx
set ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxxxxxx
set GEMINI_API_KEY=AIzaxxxxxxxxxxxxxxxx
set GROQ_API_KEY=gsk_xxxxxxxxxxxxxxxx

# Windows (PowerShell)
$env:TELEGRAM_BOT_TOKEN="7..."
$env:ANTHROPIC_API_KEY="sk-ant-..."
$env:GEMINI_API_KEY="AIza..."
$env:GROQ_API_KEY="gsk_..."

# Linux / Mac
export TELEGRAM_BOT_TOKEN=7...
export ANTHROPIC_API_KEY=sk-ant-...
export GEMINI_API_KEY=AIza...
export GROQ_API_KEY=gsk_...
```

Указывать все три ключа не обязательно — достаточно одного. Если ключ для провайдера не задан, бот сообщит об этом при попытке его выбрать.

## Запуск

```bash
python bot_client.py
```

## Что умеет бот

Пользователь выбирает раздел кнопкой или пишет вопрос свободным текстом:

| Кнопка | Что делает |
|--------|-----------|
| 📋 Документы | Протоколы, отчёты, планы, приказы |
| 🤝 Соцпомощь | Льготы, субсидии, волонтёры |
| 👥 Члены | Приём, база данных, первички |
| 🎖️ Мероприятия | 9 мая, уроки мужества, захоронения |
| 📢 Инфо и СМИ | Посты ВКонтакте, пресс-релизы, рассылки |
| 🔀 Авто | Бот сам определяет нужного агента |
| ⚙️ Провайдер ИИ | Переключение между Claude / Gemini / Groq |

По умолчанию используется Claude. Переключить провайдера можно в любой момент кнопкой «⚙️ Провайдер ИИ» — выбор сохраняется для текущего диалога.

## Где взять бесплатные ключи

- **Gemini:** aistudio.google.com/apikey → Create API key → бесплатный тариф ~1500 запросов/день
- **Groq:** console.groq.com/keys → Create API Key → бесплатный тариф с лимитом запросов в минуту
