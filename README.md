# Cleanup & Performance Assistant

Простой PowerShell-скрипт для очистки временных файлов и улучшения производительности Windows.

## Что делает
- очищает папки временных файлов
- очищает корзину
- сбрасывает DNS-кеш
- переключает план питания на `Ultimate Performance` или `High performance`

## Как использовать
1. Откройте PowerShell от имени администратора.
2. Перейдите в папку проекта:

```powershell
cd "c:\Users\HP\Documents\Project"
```

3. Запустите скрипт:

```powershell
.\cleanup-performance.ps1
```

Если политика выполнения запрещает запуск скриптов, выполните:

```powershell
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
```

## Важно
- Некоторые файлы в использовании системой не могут быть удалены.
- Скрипт не включает автоматическое обновление драйверов или сложные оптимизации.
- Перед использованием рекомендуется сохранять важные данные.
## MT4 советник для золота
Добавлен эксперт `XAU_GoldAdvisor.mq4`, который предназначен для торговли золотом на символе `XAUUSD`.

Особенности эксперта:
- торговля только на XAUUSD / GOLD;
- фильтрация по спреду;
- сигналы на основе пересечения EMA (50/200) на H1;
- ATR-основанные уровни Stop Loss и Take Profit;
- трейлинг-стоп и дневной лимит сделок.

> Этот советник создан как стартовая точка. Перед реальной торговлей обязательно тестируйте в Стратегическом тестере и подбирайте параметры под ваш брокер.
