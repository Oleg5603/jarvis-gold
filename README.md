# Advisor — Торговый советник GoldAdvisor

## О проекте

Торговый советник (Expert Advisor) для MetaTrader 4/5 по торговле золотом (XAUUSD).

## Файлы

- `GoldAdvisor.mq4` — советник на языке MQL4 для MT4

## Установка

1. Скопировать `GoldAdvisor.mq4` в папку `MT4/MQL4/Experts/`
2. В MetaTrader 4: Навигатор → Советники → GoldAdvisor
3. Перетащить на график XAUUSD и настроить параметры
