# Telegram Bot — Общая база (main)

Репозиторий всех проектов хозяина. Каждый проект — в своей ветке.

## Ветки проектов

| Ветка | Проект | Статус |
|-------|--------|--------|
| `main` | Общая база, бот Гаврика | Активен |
| `misemia` | Бот психотерапевта Светланы | Активен |
| `gavrik` | Основной бот Гаврика | Активен |
| `advisor` | Торговый советник GoldAdvisor | В разработке |
| `nutritionist` | Нутрициолог | Готова к разработке |
| `veterans` | Ветераны | Готова к разработке |

## Как работать с ветками

```bash
# Переключиться на проект
git checkout misemia

# Вернуться на main
git checkout main

# Посмотреть все ветки
git branch -a
```

## Общие файлы (main)

- `bot.py` — бот Гаврика (основной ИИ-ассистент хозяина)
- `gold_news.py` — парсер новостей по золоту
- `memory.json` — память Гаврика
- `requirements.txt` — зависимости Python
- `.env` — токены (не в репозитории, только на сервере)
