# Gavrik — Основной бот Гаврика

## О проекте

Личный ИИ-ассистент хозяина в Telegram. Помогает с текстами, кодом, анализом, разработкой приложений и сайтов, созданием квизов.

## Что умеет

- Отвечает на вопросы, помогает с задачами любой сложности
- Пишет и правит код
- Помогает анализировать данные
- Знает историю разговоров через `memory.json`
- Мониторит новости по золоту (`gold_news.py`)

## Основные файлы

| Файл | Назначение |
|------|-----------|
| `bot.py` | Основной код бота |
| `gold_news.py` | Парсер новостей по золоту |
| `memory.json` | Долгосрочная память |
| `owner.txt` | Chat ID хозяина |
| `requirements.txt` | Зависимости Python |

## Сервис на сервере

```bash
systemctl status telegram-bot
systemctl restart telegram-bot
```

## Токен

В `.env` → `TELEGRAM_BOT_TOKEN`
