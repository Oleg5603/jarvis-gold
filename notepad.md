### Идеальная комбинация из 5 скилов
1. `meta` — дирижёр (думать о том, как думать)
2. `thinking-first-principles` — фундамент (разбирать до истин)
3. `thinking-second-order` — стратег (видеть цепочку последствий)
4. `adversarial-review` — адвокат дьявола (атаковать свои выводы)
5. `thinking-systems` — архитектор (собрать в единую систему)

### Метапрограммы (описание пакета Stack Skills)
Скилы, основанные на методологиях величайших мыслителей: де Боно, Медоуз, Голдратт, Кант, Аристотель и др.
Содержит навыки для логики, креативности, системного и стратегического мышления.
Позволяет "призвать" мыслительный аппарат конкретного гения.

## План работы с компанией ЭРИС (Чайковский)

**ООО «Эрис»** — производство КИПиА, 106 сотрудников
Директор: **Юрков Владимир Иванович**
ИНН 5920017357, ул. Промышленная 8/31, Чайковский
Дочка: **Эрис КИП** (оптовая торговля оборудованием, 42 чел.)

### Что продаём
- Электронный документооборот (договоры, счета, накладные)
- Автоматизация складского учёта
- CRM / учёт заказов и отгрузок
- Интеграция Эрис ↔ Эрис КИП (единая база)
- Telegram-бот для заявок от клиентов
- Аналитика и дашборды (выручка, загрузка, брак)
- Автоматические ежедневные отчёты директору
- Электронный наряд-заказ на производстве

### Этапы
1. Позвонить Юркову В.И. — договориться на встречу
2. До встречи: найти тендеры, клиентов, поставщиков
3. Встреча-знакомство: 5 вопросов, не продавать — слушать
4. Отправить КП в течение 24 часов после встречи
5. Через 3-4 дня — звонок-follow-up
6. Пилот на одном процессе

### 5 вопросов для встречи
1. Как сейчас документооборот между Эрис и Эрис КИП?
2. Сколько времени уходит на согласование документов?
3. Есть ли 1С — какой конфигурации?
4. Какие процессы хотели бы ускорить в первую очередь?
5. Кто принимает решения по IT/автоматизации?

### Возражения
| Возражение | Ответ |
|-----------|-------|
| «Уже есть 1С» | «Отлично, от неё и оттолкнёмся — не меняем, дополняем» |
| «Дорого» | «Пилот бесплатный — сначала увидите результат» |
| «Некогда» | «Внедрение беру на себя, ваши люди тратят максимум 2 часа» |
| «Подумаем» | «Что конкретно смущает? Давайте разберём сейчас» |

### Сценарий встречи (30–40 мин)
- **0–5 мин:** установить контакт, снять защиту («не продаю — изучаю»)
- **5–20 мин:** диагностика — задаёшь вопросы, слушаешь
- **20–30 мин:** показать ценность через их боли (не цену)
- **30–40 мин:** договориться о следующем шаге (пилот или следующая встреча)

### Полный объём работ для ЭРИС (список)

**Документооборот**
- Электронное согласование договоров, счетов, актов
- Автоподстановка реквизитов
- Архив документов с поиском
- Уведомления ответственным

**Учёт и склад**
- Единая база остатков (Эрис + Эрис КИП)
- Приход / расход / инвентаризация
- Автоматический расчёт потребностей под заказы
- Учёт брака

**Производство**
- Электронный наряд-заказ (замена бумажных маршрутных карт)
- Учёт загрузки производства
- План / факт по выпуску

**CRM и продажи**
- База клиентов и заказов
- Статусы по сделкам
- История взаимодействий

**Аналитика для директора**
- Ежедневная сводка в Telegram (выручка, отгрузки, остатки)
- Дашборд план/факт
- Автоотчёты без участия менеджеров

**Интеграции**
- Интеграция с 1С (если есть)
- Синхронизация Эрис ↔ Эрис КИП
- Telegram-бот для заявок клиентов

**Итого: 8 направлений, ~20 задач**

### Сроки разработки

| Блок | Срок |
|------|------|
| Документооборот (пилот) | 2 недели |
| Учёт и склад | 4–6 недель |
| Производство (наряды, план/факт) | 4–6 недель |
| CRM и продажи | 3–4 недели |
| Аналитика + Telegram-сводки | 1–2 недели |
| Интеграции (1С, синхронизация) | 2–4 недели |

**Итого: 4–5 месяцев** последовательно. Параллельно — 2,5–3 месяца.

### Экономия для ЭРИС

| Что исчезает | Сколько |
|-------------|---------|
| Ручное согласование документов | −2 ч/день на менеджера |
| Поиск в бумажном архиве | −30–40 мин/день |
| Инвентаризация склада | с 3 дней → 4 часа |
| Ежемесячные сводки директору | с 8 ч → автоматически |
| Ошибки двойного учёта Эрис↔Эрис КИП | практически до нуля |

**В деньгах:** 10–15 задействованных сотрудников → экономия ~40–80 часов/мес → **80–160 тыс. руб./мес**.
Окупаемость автоматизации — **2–4 месяца**.
