# Блокнот Гаврика

## Stack Skills и мета-навыки для Claude Code

### 5 рекомендованных пакетов

| # | Пакет | Что даёт |
|---|-------|---------|
| 1 | **Stack Skills** (нейрофу/stack-skills) | Фреймворки: OODA Loop, Pre-Mortem, WRAP, Cynefin, SCAMPER, Шесть Шляп |
| 2 | **cc-thinking-skills** (tjboudreaux) | Скилы: first-principles, second-order, systems thinking |
| 3 | **Meta Skill for Claude Code** (nicknisi) | Мета-контроль: планирование сессии, темп, журнал ошибок |
| 4 | **skills-for-humanity** | Гуманитарные фреймворки, этика, коммуникация |
| 5 | **neurofoo/agent-skills** | Агентские паттерны: декомпозиция, рефлексия, адаптация |

### Идеальная комбинация из 5 скилов
1. `meta` — дирижёр (думать о том, как думать)
2. `thinking-first-principles` — фундамент (разбирать до истин)
3. `thinking-second-order` — стратег (видеть цепочку последствий)
4. `adversarial-review` — адвокат дьявола (атаковать свои выводы)
5. `thinking-systems` — архитектор (собрать в единую систему)

### Метапрограммы (описание пакета Stack Skills)
Скилы, основанные на методологиях величайших мыслителей: де Боно, Медоуз, Голдратт, Кант, Аристотель и др.
Содержит навыки для логики, креативности, системного и стратегического мышления.
Позволяет "призвать" мыслительный аппарат конкретного гения.

### Как создать свой скил
```bash
mkdir -p ~/.claude/skills/название-скила
# создать файл SKILL.md с frontmatter и инструкцией
```
